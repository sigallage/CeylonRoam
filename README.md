CeylonRoam is a travel planning platform designed to help people plan trips in Sri Lanka faster and with more confidence. It brings itinerary planning, route optimization, and language assistance into one experience so travelers can spend less time organizing and more time enjoying the trip.

What Problem It Solves
Planning a trip usually means:
- Searching many places to decide where to go
- Manually arranging locations in a sensible order
- Estimating travel time between places (often inaccurately)
- Dealing with language barriers when communicating or understanding local context

CeylonRoam reduces this friction by generating plans, organizing the travel sequence, and supporting translation features in one place.

Core Features
1) Personalized Itinerary Generation
- Creates a multi-day travel plan based on the traveler’s purpose and preferences
- Produces a clear day-by-day structure (morning/afternoon/evening)
- Includes practical tips and a budget-aware mindset for planning

2) Smart Route Optimization
- Reorders selected destinations into an efficient visiting sequence
- Helps reduce unnecessary travel and time wasted between locations
- Can account for real-world road conditions (where available)

3) Voice + Language Assistance
- Converts speech to text for easier communication and note taking
- Translates text across supported languages (useful for travelers and locals)
- Helps users handle common travel interactions more comfortably

4) User Accounts & Saved Plans
- Users can create accounts and log in
- Itineraries can be saved and revisited later
- Supports password recovery to keep access reliable

Who It’s For
- Solo travelers planning their first Sri Lanka trip
- Couples and families who want structured travel days
- Groups coordinating multiple interests and destinations
- Anyone who wants faster planning and less logistical stress

What Makes It Different
- Combines itinerary creation, route ordering, and translation in one product
- Focuses on Sri Lanka travel context rather than generic planning
- Supports both planning and “on-the-trip” assistance

Typical User Journey
1) User shares trip preferences (interests, budget, travel dates)
2) The app generates a proposed itinerary
3) The user selects or edits destinations they want
4) The app optimizes the travel order to reduce wasted time
5) The user saves the plan and uses it during the trip
6) Voice translation helps with communication when needed

Project Status
CeylonRoam is structured as a scalable platform with multiple specialized services behind a single user experience. The project is designed to be deployable and extensible as new travel features are added.
